# Rockstar Games - Senior Frontend Engineer (Online Services)

## Job Details
- **Location:** Rockstar Toronto (Oakville, ON)
- **Type:** Full-time, In-office
- **Department:** Online Services
- **Pay Range:** $96,400 - $142,660 CAD

## Job Description
Rockstar Games is seeking a passionate and experienced Frontend Engineer with a focus on the "back of the frontend" to join their online development team.

### What We Do
- Create connected experiences across game franchises
- Architect and maintain high-impact features for Rockstar community and websites
- Create innovative in-game experiences, including user-generated content
- Build services and tools supporting consumer and internal needs

### Responsibilities
- Development of accessible front-end components and applications using React
- Integrating front-end applications with Next.js and ASP.NET Core (C#)
- Develop comprehensive test suites
- Web performance optimization
- Collaboration with Rockstar technology teams worldwide

### Requirements
- Degree in computer science OR 5+ years of professional web development
- 5+ years of experience using React
- 5+ years of experience using Node.js or C#
- Extensive experience in accessible web development
- Ability to communicate effectively with global teams
- Experience with automated testing, shared code, and Git
- Knowledge of Agile, software development, and web-security best practices
- Passion for web technologies and staying current

### Pluses
- Experience with high-traffic, responsive websites
- Next.js SSR/ISR experience
- Exposure to Elasticsearch/OpenSearch
- Exposure to observability tools
- Knowledge of Rockstar's game titles

### Application Process
Apply with a resume and cover letter demonstrating how you meet the skills above. Rockstar is committed to equal opportunity and will provide reasonable accommodations.
