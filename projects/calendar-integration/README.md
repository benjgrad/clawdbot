# Calendar Integration Project

## Overview
This project provides a comprehensive calendar integration system with advanced features for event management, goal tracking, and personal reflection.

## Key Features
- Multi-platform OAuth authentication
- Intelligent event tracking and classification
- Goal alignment and progress tracking
- Daily reflection integration

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Configure OAuth credentials
3. Run initial setup: `python -m calendar_integration.setup`
